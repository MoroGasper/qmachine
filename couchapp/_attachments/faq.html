Frequently Asked Questions
What browser should I use?
We recommend Google Chrome highly, and Safari, Opera, and Firefox should also work. Internet Explorer lacks some of the advanced features that Quanah requires, though, so if you're using IE, things probably won't work. We are very interested, however, in hearing about users' experiences with the Chrome Frame plugin for IE, which may fix those issues.
