QMachine

QMachine is a web service that uses Quanah to act as a "supercomputer" for running scientific computations using only web browsers and CouchDB. I have also written a "port" to Node.js that uses SQLite for its persistent storage mechanism. I am currently writing several papers about Quanah and QMachine, but I promise I will write more documentation soon!

There are several "mirrors" available on Bitbucket, GitHub, and Google Code.

A Node.js module can be installed via NPM.
