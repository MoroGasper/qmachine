Frequently Asked Questions
What browser should I use?
We recommend Google Chrome highly for its advanced feature support and built-in JavaScript console. Firefox and Opera work with Quanah just fine, although neither has an interactive console by default. If you are willing to install a plugin to get the console, we recommend Firebug for Firefox. Strangely, Safari has an excellent built-in console (identical to Chrome's), but it lacks a few features necessary for volunteering. Finally, Internet Explorer lacks many advanced features that are present in modern browsers, including some required by Quanah. We are investigating Google's new Chrome Frame plugin to see if it patches IE suitably, but we do not currently recommend Internet Explorer for use with Quanah.
